# Dashboard Kemiskinan Provinsi Indonesia 2024
> Dashboard visualisasi persentase penduduk miskin per provinsi di Indonesia berdasarkan data BPS 2024.

🌐 Demo: [https://final-project-visdat.vercel.app](https://final-project-visdat.vercel.app/)

## Isi Dashboard
- Chart 1: Bar Chart — persentase kemiskinan total per provinsi (Sept 2024)
- Chart 2: Doughnut Chart — rata-rata kemiskinan per kelompok pulau
- Chart 3: Line Chart — tren kemiskinan perkotaan vs perdesaan (Maret vs Sept 2024)
- Fitur interaktif: tooltip hover, filter dropdown per pulau, toggle dataset
- Animasi: entrance Chart.js, count-up number KPI, CSS fade-in

## Sumber Data
- Nama dataset: Persentase Penduduk Miskin (P0) Menurut Provinsi dan Daerah 2024
- URL sumber: https://www.bps.go.id/id/statistics-table/2/MTkylzl=/persentase-penduduk-miskin--p0--menurut-provinsi-dan-daerah.html

## Cara Jalankan di Lokal
# Jalur A (static):
Buka index.html langsung di browser (atau pakai Live Server di VS Code)
